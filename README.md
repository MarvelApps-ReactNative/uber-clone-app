# Uber Clone App
### About the Project
We have created uber app Clone, Uber is the most popular app in the market, where those who drive and deliver can connect with riders. In cities where Uber is available, you can use the Uber app to request a ride. When a nearby driver accepts your request, the app displays an estimated time of arrival for the driver heading to your pickup location which help the user to save the time and Uber's advantages include door-to-door convenience, safety, and reliable quality.
